About This Test

This demo page shows how the green navigation bar A/B test would appear on a website. The JavaScript and CSS files will change the navigation background from the current dark gray to green.

Test Variants

Control: Original dark gray navigation bar (#1f2937)

Treatment: Green navigation bar (#22c55e)

How to Test

1. Load this page to see the control version

2. Run the variant.js file to see the treatment version

3. The JavaScript will automatically apply the green styling
